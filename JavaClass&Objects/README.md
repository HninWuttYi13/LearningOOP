# Java vs JavaScript OOP (Day 3)

## 🔹 Class Structure

### Java

- Everything must be inside a class
- Strong structure
- Requires types

### JavaScript

- Flexible
- Class is optional
- Uses prototype internally

---

## 🔹 Object Creation

### Java

```java
Professor walsh = new Professor("Walsh", "Psychology");
```

### JavaScript

```javascript
const walsh = new Professor("Walsh", "Psychology");
```
